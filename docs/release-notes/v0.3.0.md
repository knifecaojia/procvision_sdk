# v0.3.0

## 重大变更（Hard break）
- `execute` 入参硬切为新结构：
  - `execute(step_index, step_desc, cur_image, guide_image, guide_info)`
  - 不再接收 `session/user_params/shared_mem_id/image_meta`
- 适配器协议 `call.data` 硬切：
  - 新增：`step_desc`、`guide_info`
  - 新增：`cur_image_shm_id/cur_image_meta`、`guide_image_shm_id/guide_image_meta`
  - 移除旧字段示例：`shared_mem_id/image_meta/user_params/session`
- CLI `run` 参数硬切：
  - 新增：`--cur-image`、`--guide-image`（`--image` 作为 guide-image 兼容别名）
  - 新增：`--step-desc`、`--guide-info`（支持 `@file.json`）
  - 移除：`--params`

- 见 `docs/migration_guide_v0.3.0.md`
