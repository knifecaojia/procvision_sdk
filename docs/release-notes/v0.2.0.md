# v0.2.0

## 重大变更（Hard break）
- 算法接口硬切为 Execute-only：
  - 移除 `get_info`
  - 移除 `pre_execute`
  - 移除全部生命周期与钩子（`setup/teardown/reset/on_step_*`）
- 适配器协议硬切为单次调用：
  - `call` 不再使用 `phase`
  - `hello.capabilities` 仅保留 `execute`
- CLI 行为调整：
  - `run/validate` 仅驱动 `execute`
  - 移除 legacy 分支与相关参数

## 文档
- 新规范：`spec.md`、`protocol_adapter_spec.md`、`runner_spec.md` 均已对齐 Execute-only。
- 新迁移指南：`docs/migration_guide_v0.2.0.md`
